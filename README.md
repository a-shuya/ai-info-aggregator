# AI情報収集ページ

AI・ビジネス・技術記事の自動収集・表示サイト + 市況・天気ダッシュボード

## 概要

複数のWebサイトの公式RSSフィードから最新記事を自動収集し、日付別・サイト別に整理して表示するWebアプリケーションです。記事閲覧に加え、株価・為替・天気情報のダッシュボード機能も提供します。

## 主要機能

### 📰 記事一覧機能
- **3列固定グリッド**: PC版で美しい3列レイアウト、モバイルで1列表示
- **新着順ソート**: 全期間・全タブで最新記事が上に表示
- **サイト別フィルタ**: 20サイトの記事をタブで切り替え
- **期間選択**: 本日・昨日・一昨日・月別表示
- **元サイトリンク**: 各サイトタブ選択時に元サイトへのリンクを表示

### 📊 市況・天気ダッシュボード  
- **株価・為替**: 日経平均・ドル円・S&P500の前日終値と変動率
- **天気予報**: 東京3日間予報（3時間ごと、☀️☁️🌧️記号付き）
- **リアルタイム**: Yahoo Finance API・気象庁API使用

### 🏷️ タグ管理システム
- **分割保存**: 各サイト別にJSONファイル管理
- **1年間蓄積**: URL重複排除で過去1年分のデータ保持
- **自動拡張**: 新サイト追加時の自動対応

### 🎨 UI/UX機能
- **ホバー効果**: 記事カードの立体的アニメーション
- **ファビコン表示**: 各記事にサイトアイコン表示
- **レスポンシブ**: PC・タブレット・モバイル最適化

### ホスティング・自動化

- **GitHub**: ソースコード管理・Actions実行

## ローカル開発

### 必要な環境
- Python 3.11+
- Node.js 18+

### セットアップ

1. リポジトリのクローン
```bash
git clone <repository-url>
cd ai-info-aggregator
```

2. Python 依存関係のインストール
```bash
pip install -r requirements.txt
```

3. RSS データの収集
```bash
python rss_collector.py
```

4. Node.js 依存関係のインストール
```bash
npm install
```

5. 開発サーバーの起動
```bash
npm run dev
```

## デプロイ・運用

### 自動化フロー
1. **GitHub Actions**: 毎時0分（UTC）にRSS収集実行
2. **データ更新**: タグ別JSONファイル（`data/tags/*.json`）を更新
3. **自動コミット**: 変更をGitHubリポジトリにプッシュ
4. **Vercel自動デプロイ**: プッシュ検知で自動的にサイト更新

### Vercel設定
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **連携**: GitHub リポジトリ自動デプロイ

### データ管理
- **保存場所**: `data/tags/[サイト名].json`（各サイト別）
- **保存期間**: 1年間（URL重複排除）
- **更新頻度**: 毎時（JST 9分）
